radio buttons

Radio buttons allow the user to select one and only one option from a list.

Checkboxes

Checkboxes work similarly to radio buttons, but allow users to select multiple options. Because we want to be able to collect all of a user's selections, we will first need to learn about something called an array, which we will do shortly. In its simplest form, an array is simply a list of multiple items packaged together.
